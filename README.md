# 🛍️ React Mini Store

A modern e-commerce web application built using **React.js**.  
This project demonstrates a complete frontend shopping experience including cart management and a checkout flow.

---

## 🌐 Live Demo

👉 https://react-ecommerce-store-sable.vercel.app/

---

## 🚀 Features

- 🏠 Home page with hero section and featured products  
- 🛒 Add to Cart with quantity management  
- ➕ Increase / ➖ Decrease product quantity  
- ❌ Remove items from cart  
- 💾 Persistent cart using localStorage  
- 🧾 Checkout page with order summary  
- ✅ Order success page  
- 🎯 Clean and responsive UI  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Routing:** React Router  
- **State Management:** useState, useEffect  
- **Styling:** CSS  
- **API:** Fake Store API
